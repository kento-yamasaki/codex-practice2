# 洋ちゃれのカフェ preview

洋ちゃれのカフェのトップページをローカルで確認するためのシンプルなHTTPサーバーです。Python以外の依存は不要です。

## サーバーの起動
1. Python 3 がインストールされていることを確認します。
2. リポジトリ直下で次のコマンドを実行します。
   ```bash
   python serve.py
   ```
3. ブラウザで http://localhost:8000 を開きます。

### ポートを変えたい場合
```bash
python serve.py --port 3000
```

起動後は `Ctrl+C` で終了できます。
